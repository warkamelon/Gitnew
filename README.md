# УЧЕТ СТАТЕЙ В ЖУРНАЛЕ
## Какие задачи решает система:
1. Добавление и хранение статей
2. Создание удобного для читателя интерфейса с поиском и фильтрацией статей
## Пользователи:
- Авторы статей - отправляют статьи
- Администратор – принимает/отклоняет заявки на публикацию
- Читатели - читают статьи, оценивают статьи
## Основные функции:
1. Управление статьями
- Добавление статьи: внесение данных (название, автор, аннотация, дата подачи, статус статьи, номер статьи).
- Удаление статьи
2. Управление авторами
- Добавление и редактирование данных об авторах (ФИО, контактные данные).
- Отслеживание количества публикаций каждого автора.
3. Статусы статей
- Статусы:  «На рассмотрении», «Принята», «Отклонена», «Опубликована».
4. Поиск и фильтрация
- Поиск по названиям, авторам, номеру статьи.
- Фильтры по статусам, датам подачи, тематикам, рейтингу.
5. Оценка статей читателем (по десятибалльной шкале)
## Таблицы:
- Пользователи: имя аккаунта, пароль, количество статей, почта
- Журналы: название, количество статей в журнале, тематика
- Статьи: название, номер, рейтинг, статус, автор, журнал, тематика
